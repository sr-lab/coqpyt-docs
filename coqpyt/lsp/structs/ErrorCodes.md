# `ErrorCodes`

The error codes used by the JSON RPC to identify different errors that could occur on the server.

## Values

```python
ParseError = -32700
InvalidRequest = -32600
MethodNotFound = -32601
InvalidParams = -32602
InternalError = -32603
serverErrorStart = -32099
serverErrorEnd = -32000
ServerTimeout = -32004
ServerQuit = -32003
ServerNotInitialized = -32002
UnknownErrorCode = -32001
RequestCancelled = -32800
ContentModified = -32801
```
