# `InvalidFileException(Exception)`

An error created when a file that does not exist is attempted to be accessed.

## Attributes

`file : str`
> The path to the file that was attempted to be accessed.


## Operations
```python
__init__(self, file: str)
```
> Creates a new InvalidFileException instance.
> 
> `file : str`
> > The path to the file that was attempted to be accessed.
